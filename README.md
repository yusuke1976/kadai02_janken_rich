## ⑤工夫した点・こだわった点

- 「対戦スタート」ボタンを押すだけで、全ての動作が起こるようにしました
- 対戦結果を、音声で読み上げるようにしました
- 「あいこ」以外の場合、画面全体が揺れて、勝った手も揺れるようにしました
- 勝った回数をカウントするスコアを追加しました

## ⑥難しかった点・次回トライしたいこと(又は機能)

- ボタン１つで、日本語バージョン⇔英語バージョンに切り替えたかったのですが、難しくてできませんでした
